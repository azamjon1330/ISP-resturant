# YouIt Café — Запуск проекта

## Структура проекта

```
YouIt/
├── backend/          ← Go API (порт 8080)
├── frontend/         ← React + Vite (порт 3000)
├── mobile/           ← React Native Expo
├── init.sql          ← PostgreSQL схема
├── docker-compose.yml
└── .env
```

---

## 🚀 Запуск через Docker (рекомендуется)

### Требования
- [Docker Desktop](https://www.docker.com/products/docker-desktop/)

### Команды
```bash
# Из папки проекта
docker compose up --build

# Остановить
docker compose down
```

После запуска:
- Касса / Кухня / Админ: http://localhost:3003
- API: http://localhost:8080

**На VPS (109.123.253.238):**

- API: http://109.123.253.238:8080

## 💻 Локальный запуск

### Backend (Go)
```bash
# Установить Go: https://golang.org/dl/
cd backend
go mod tidy
go run ./cmd/main.go
```

### Frontend
```bash
cd frontend
npm install
npm run dev
# Открыть: http://localhost:3000
```

### Mobile (React Native Expo)
```bash
# Установить Expo CLI
npm install -g expo-cli

cd mobile
npm install

# Запуск
npx expo start

# Сканировать QR через приложение Expo Go (Android/iOS)
```

**Важно для мобильного:** В файле `mobile/src/api/index.js` замените `YOUR_SERVER_IP` на IP вашего компьютера в локальной сети.

## 🎴 Реферальная система

1. В Кассе (или Админ → Агенты) создаётся реферальный агент
2. Агент получает **7-значный уникальный код**
3. Генерируются карточки: 1 золотая + N обычных (по умолчанию 20)
4. Каждая карточка имеет **QR-код**
5. При сканировании QR на кассе — скидка применяется автоматически
6. **Золотая карта:** после N визитов (по умолчанию 10) — скидка на следующий заказ
7. **Обычные карты:** скидка 5% каждый раз; после M использований всех карт — бонус владельцу золотой карты
8. В мобильном приложении агент вводит 7-значный код → видит своё меню, карты с QR, историю
